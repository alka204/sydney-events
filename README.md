# Sydney Events

A full-stack web application for managing events and tickets.

## Features
- Admin dashboard
- Event creation and management
- Ticket booking
- User authentication

## Tech Stack
- Backend: Node.js, Express, MongoDB, Passport.js
- Frontend: React, Vite, Tailwind CSS

## Setup
1. Clone the repo
2. Run `npm install` in both backend and frontend
3. Set `.env` files with your credentials
4. Run `npm start` (or `npm run dev` for frontend)
